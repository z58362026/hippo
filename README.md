# hippo

## 一、添加项目代码片段

项目中，.vscode 目录，右键 create snippet，在 maker 平台查找模版 id

## 二、代码添加注释

选中代码，执行 cmd+t，或者 shift+cmd+p,输入 chatgpt comment

## 三、生成代码

输入注释，执行 cmd+g, 或者 shift+cmd+p,输入 chatgpt generate

## 四、代码优化

选中代码，执行 cmd+o, 或者 shift+cmd+p,输入 chatgpt optimization

## 五、查询内容

点击左侧导航，可以在线查询内容。

## 注意

本插件是实验阶段，暂不准确
